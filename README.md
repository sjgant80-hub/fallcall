# FallCall

**7-Layer Sales Call Psychology Engine**

Sovereign single-file HTML tool for sales call analysis. Paste a transcript, get deep psychological analysis across 7 layers.

## Features

- 7-layer analysis: Surface → Intent → Psychology → Power → Risk → Opportunity → Strategy
- 10 objection codes (OBJ-01 to OBJ-10) with counter-strategies
- 8 buying signal codes (SIG-01 to SIG-08)
- Register tracking (REG-0 to REG-4) — maps rapport depth
- FreudEngine psychology layer
- Comparison table vs Gong/Chorus/Revenue.io
- Multi-provider AI: Gemini (free) → DeepSeek (free) → Anthropic (paid)
- Dark aesthetic, zero dependencies, runs in browser

## Run

Double-click `FallCall.html`. That's it. Sovereign.
